# ############# Windows Server Configuration #####


This example demonstrates common administrative tasks: installing features, managing users, and ensuring services are running

Ref win_server_config.yml    # For complete win_server_configuration

Essential Windows Modules
Module 	Purpose
win_feature	Installs/uninstalls Windows Server roles and features.
win_user	Manages local Windows user accounts.
win_package	Installs software using MSI or EXE installers from a URL or local path.
win_updates	Checks for, downloads, and installs Windows updates.
win_regedit	Adds, edits, or removes Windows Registry keys and values.
